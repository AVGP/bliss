Bliss.js

Heavenly JavaScript

  • Only deal with native DOM elements and arrays, no weird wrapper objects
  • Ultra-lightweight — only 1.5KB minified & gzipped!
  • No pollution of native prototypes, only adds a _ property on elements & arrays
  • Super easy to extend

You don’t want to use a big, bloated framework but find native DOM APIs are often a bit clumsy? Bliss fixes that, with a tiny footprint and without locking you in.
